# Webscraping Go

Web scraping example using golang and goquery.

[What is goquery?](https://godoc.org/github.com/PuerkitoBio/goquery)

# Requirement

- Go: 1.14

# Installation

Install Dependency Packages.

```bash
$ go get
```

# Usage

Execute.

```bash
$ go run main.go
```
